PatchPush

PatchPush lets you receive automatic SMS notifications when Steam games release new updates.

Opt-In Process

Users opt in by entering their mobile phone number on this website (coming soon). A verification code is sent to confirm ownership of the phone number. After confirmation, users can manage which games they want update alerts for.

What Messages Contain

SMS notifications include: – Game title – New version number – Brief update description – Optional link to official patch notes

PatchPush is in early development.
